# ruscup_aurora

A Drupal 7 theme based on Aurora and leveraging SASS.

Created for the http://russian-cup.ru website: a ballroom dancing tournament.

## Requirements

* Drupal 7
* [Aurora][1] base theme
* To effectively edit CSS, you need a SASS compiler, e. g. [Compass][2].
